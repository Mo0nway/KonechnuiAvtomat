# KonechnuiAvtomat
Детерминированный конечный автомат для процесса варки зелья
Рассмотрим детерминированный конечный автомат (ДКА), моделирующий процесс варки зелья в игровой среде. Особенность ДКА — однозначность переходов: каждое действие строго задаёт переход между состояниями.

Входной алфавит Σ={B,H,M,S,T}:

B — базовая основа (Base);

H — ингредиент‑трава (Herb);

M — минерал (Mineral);

S — помешивание (Stir);

T — нагревание (Temperature).

Выходной алфавит не требуется: автомат выполняет функцию распознавания корректной последовательности действий.
